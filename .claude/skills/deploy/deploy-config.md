# Deploy Config Reference

## Ambientes

- `dev`: feedback rápido e validações básicas.
- `qa`: integração, testes funcionais e validação de release.
- `prd`: estabilidade, segurança, observabilidade e rollback definidos.

## Requisitos Minimos

- Imagem versionada.
- Configuração externa ao código.
- Health checks funcionais.
- Telemetria habilitada.
- Estratégia de rollback documentada.
